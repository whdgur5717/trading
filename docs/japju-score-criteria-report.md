# 잡주력 계산 기준 보고서

## 1. 결론

잡주력은 공식 금융 분류가 아니라 제품용 점수입니다. 이 보고서에서는 잡주력을 아래처럼 정의합니다.

> 잡주력 = 작고, 거래가 얇고, 가격이 심하게 흔들리고, 시장보다 과하게 움직이며, 종목 상태/재무/공시 신호가 불안한 정도

100점 만점으로 계산합니다.

| 기준                | 배점 | 왜 잡주 기준으로 쓰는가                                                                   | 핵심 데이터                           |
| ------------------- | ---: | ----------------------------------------------------------------------------------------- | ------------------------------------- |
| 가격 급등락/변동성  |   25 | 잡주라고 불리는 종목은 짧은 기간 가격이 크게 흔들리는 경우가 많음                         | KIS 일봉, 금융위 주식시세             |
| 거래 얇음/거래 이상 |   20 | 거래가 얇으면 작은 매수/매도에도 가격이 크게 움직이고, 거래 급증은 과열 신호가 될 수 있음 | 금융위 거래량/거래대금, `stocks.json` |
| 소형성              |   15 | 작은 회사일수록 정보/수급/유동성에 취약하고 변동성이 커지기 쉬움                          | 금융위 시가총액, `stocks.json`        |
| 시장 대비 과민 반응 |   10 | 시장 전체가 흔든 것인지, 해당 종목만 유난히 튄 것인지 구분해야 함                         | 금융위 지수시세, 종목 수익률          |
| 종목 상태 플래그    |   15 | 관리종목, 거래정지, 시장경고는 투자유의가 필요한 직접 신호                                | `stocks.json`                         |
| 재무/공시 불안      |   15 | 적자, 자본잠식 후보, 정정/철회 공시는 회사 신뢰도와 위험을 낮춤                           | OpenDART                              |

## 2. 가격 급등락/변동성

| 항목        | 기준                                                                | 점수 | 근거                                                                  | 간단 예시                                    |
| ----------- | ------------------------------------------------------------------- | ---: | --------------------------------------------------------------------- | -------------------------------------------- |
| 종가 변동성 | 최근 20거래일 일간 등락률 표준편차를 같은 시장 종목들과 비교        |   10 | 금융에서 변동성은 가격/수익률이 얼마나 흩어지는지 보는 대표 위험 지표 | 코스닥 상위 10% 변동성이면 높게 가산         |
| 장중 고저폭 | 최근 20거래일 평균 `(고가 - 저가) / 종가`를 같은 시장 종목들과 비교 |    7 | 종가만 보면 장중 급등락 후 제자리로 온 종목을 놓침                    | 종가는 +1%지만 장중 고저폭이 18%면 튄 종목   |
| 급등락일 수 | 최근 20거래일 중 `abs(일간 등락률) >= 10%`인 날 수                  |    8 | 한국 시장경보에서도 종가 급변, 단기 급등을 투자주의 신호로 봄         | 20일 중 3일 이상 10% 넘게 움직이면 강한 가산 |

점수화:

| 상태                                     | 점수 방향                       |
| ---------------------------------------- | ------------------------------- |
| 같은 시장 하위~중간권 변동성             | 거의 가산하지 않음              |
| 같은 시장 상위 30%                       | 약하게 가산                     |
| 같은 시장 상위 10%                       | 강하게 가산                     |
| 같은 시장 상위 5% 또는 급등락일 3회 이상 | 가격 튐을 핵심 잡주 신호로 판단 |

## 3. 거래 얇음/거래 이상

| 항목            | 기준                                                      | 점수 | 근거                                                                           | 간단 예시                                              |
| --------------- | --------------------------------------------------------- | ---: | ------------------------------------------------------------------------------ | ------------------------------------------------------ |
| 거래대금 얇음   | 같은 시장에서 거래대금 하위 백분위                        |    8 | 거래가 얇으면 원하는 가격에 사고팔기 어렵고 작은 주문에도 가격이 움직이기 쉬움 | 코스닥 거래대금 하위 10%면 강한 가산                   |
| 거래량 급증     | 당일 회전율 또는 거래량이 최근 20일 중앙값 대비 몇 배인지 |    7 | 시장경보는 가격 급변뿐 아니라 거래량 조건도 함께 봄                            | 평소 10만주 거래되던 종목이 100만주 거래되면 이상 신호 |
| 저유동성 플래그 | `stocks.json.isLowLiquidity`                              |    5 | 이미 종목 마스터에 저유동성으로 분류된 경우 직접 반영                          | 저유동성 종목이면 기본 가산                            |

점수화:

| 상태                                  | 점수 방향             |
| ------------------------------------- | --------------------- |
| 거래대금 같은 시장 하위 10%           | 유동성 위험 크게 가산 |
| 거래대금 하위 30%                     | 유동성 위험 일부 가산 |
| 거래량/회전율 최근 20일 대비 3배 이상 | 거래 과열 가산        |
| 거래량/회전율 최근 20일 대비 5배 이상 | 거래 이상 강하게 가산 |

## 4. 소형성

| 항목                 | 기준                                                    | 점수 | 근거                                                           | 간단 예시                       |
| -------------------- | ------------------------------------------------------- | ---: | -------------------------------------------------------------- | ------------------------------- |
| 시가총액 하위 백분위 | 같은 시장에서 `mrktTotAmt` 또는 `marketCap` 하위 백분위 |   15 | 소형/마이크로캡은 대형주보다 변동성, 유동성 문제가 커지기 쉬움 | 코스닥 시총 하위 5%면 강한 가산 |

점수화:

| 같은 시장 시총 위치 | 점수 방향          |
| ------------------- | ------------------ |
| 하위 5%             | 최대로 가산        |
| 하위 10%            | 강하게 가산        |
| 하위 20%            | 일부 가산          |
| 중위권 이상         | 거의 가산하지 않음 |

주의:

ETF, ETN, 우선주, REIT는 일반 보통주와 성격이 다르므로 같은 방식으로 잡주력을 계산하지 않습니다. 별도 카테고리로 분리하거나 보정합니다.

## 5. 시장 대비 과민 반응

| 항목             | 기준                                    | 점수 | 근거                                                      | 간단 예시                                           |
| ---------------- | --------------------------------------- | ---: | --------------------------------------------------------- | --------------------------------------------------- |
| 시장 대비 변동성 | 종목 변동성 / 같은 시장 대표지수 변동성 |   10 | 시장 전체가 흔든 것과 종목만 유난히 흔든 것을 구분해야 함 | 코스닥이 2% 흔들릴 때 종목이 12% 흔들리면 과민 반응 |

점수화:

| 상태                                | 점수 방향     |
| ----------------------------------- | ------------- |
| 종목 변동성이 시장지수와 비슷함     | 가산하지 않음 |
| 종목 변동성이 시장지수의 1.5배 이상 | 일부 가산     |
| 종목 변동성이 시장지수의 2배 이상   | 강하게 가산   |
| 종목 변동성이 시장지수의 3배 이상   | 최대로 가산   |

비교 기준:

| 종목 시장 | 비교 대상                                                       |
| --------- | --------------------------------------------------------------- |
| KOSPI     | 코스피 지수                                                     |
| KOSDAQ    | 코스닥 지수                                                     |
| KONEX     | 대표 지수 미확인 상태라 시장 대비 항목 제외 또는 별도 기준 필요 |

## 6. 종목 상태 플래그

| 항목          | 기준                           | 점수 | 근거                                                      | 간단 예시                        |
| ------------- | ------------------------------ | ---: | --------------------------------------------------------- | -------------------------------- |
| 관리종목      | `isUnderAdministration=true`   |    8 | 관리종목은 투자유의사항으로 별도 관리됨                   | 관리종목이면 바로 위험 신호      |
| 거래정지      | `isTradingHalted=true`         |    8 | 거래정지는 투자자 보호/시장관리를 위한 직접 조치          | 거래정지 종목이면 강한 위험 신호 |
| 시장경고/주의 | `warningLevel`이 정상값이 아님 |  4~7 | 시장경보제도는 투자주의, 투자경고, 투자위험 단계로 운영됨 | 투자경고/위험 단계면 강한 가산   |

이 항목은 합산 후 15점으로 cap을 둡니다. 같은 종목이 여러 상태에 걸려도 점수가 과도하게 폭발하지 않게 하기 위함입니다.

## 7. 재무/공시 불안

| 항목                         | 기준                                                         | 점수 | 근거                                                                  | 간단 예시                        |
| ---------------------------- | ------------------------------------------------------------ | ---: | --------------------------------------------------------------------- | -------------------------------- |
| 적자 지속                    | 최근 사업보고서/분기보고서에서 영업이익 또는 순이익 적자     |    5 | 실적 부진은 회사 안정성 리스크                                        | 2년 연속 영업손실이면 가산       |
| 자본잠식 후보                | 자본총계가 자본금보다 작거나 자본총계가 음수                 |    5 | 자본잠식은 관리종목/상장폐지 리스크와 연결될 수 있음                  | 자본총계가 음수면 강한 가산      |
| 부채비율 과다                | 부채총계 / 자본총계가 과도하게 높음                          |    3 | 재무 레버리지 과다는 재무 위험 신호                                   | 부채비율 300% 이상이면 일부 가산 |
| 정정/철회/주요사항 공시 빈도 | OpenDART `report_nm`, `rm` 기준 정정/철회/주요사항 공시 횟수 |    5 | 불성실공시는 공시불이행, 공시번복, 공시변경 같은 신뢰도 문제와 연결됨 | 최근 1년 정정/철회가 잦으면 가산 |

이 항목도 합산 후 15점으로 cap을 둡니다.

## 8. 최종 판정 문구

|   총점 | 판정            | 사용자에게 보여줄 느낌               |
| -----: | --------------- | ------------------------------------ |
|   0~24 | 정상권          | 잡주력 낮음                          |
|  25~49 | 약간 잡내       | 변동성/거래 신호가 조금 있음         |
|  50~69 | 잡주 의심       | 가격, 거래, 규모 중 여러 신호가 겹침 |
|  70~84 | 잡주력 높음     | 급등락/유동성/상태 신호가 뚜렷함     |
| 85~100 | 인간지표급 잡주 | 여러 위험 신호가 동시에 터짐         |

## 9. 예시

| 예시 종목 상태                                                                                       | 계산 결과 |
| ---------------------------------------------------------------------------------------------------- | --------- |
| 시총 상위권, 거래대금 풍부, 최근 20일 변동성 낮음, 공시/재무 문제 없음                               | 0~24점    |
| 코스닥 중소형주, 변동성 상위 20%, 거래량이 가끔 튐, 상태 플래그 없음                                 | 25~49점   |
| 코스닥 소형주, 최근 20일 중 10% 이상 급등락 2회, 거래대금 하위 20%, 시총 하위 10%                    | 50~69점   |
| 소형주, 거래대금 얇음, 변동성 상위 5%, 시장 대비 3배 이상 흔들림, 관리종목 또는 시장경고 플래그 있음 | 70점 이상 |
| 거래정지/관리종목/재무불안/공시정정이 겹치고 가격까지 급등락                                         | 85점 이상 |
