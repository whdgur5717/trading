# 트리아지 라벨

스킬이 말하는 트리아지 상태와 GitHub Issue 라벨의 대응표다. 라벨 문자열은
GitHub에 붙는 실제 값이므로 임의로 번역하지 않는다.

| 상태               | 실제 라벨         | 의미                                             |
| ------------------ | ----------------- | ------------------------------------------------ |
| 분류 필요          | `needs-triage`    | 아직 유지보수자가 검토하지 않은 이슈             |
| 정보 필요          | `needs-info`      | 작성자에게 추가 정보가 필요한 이슈               |
| 에이전트 작업 가능 | `ready-for-agent` | 요구사항이 충분해서 에이전트가 맡을 수 있는 이슈 |
| 사람 작업 필요     | `ready-for-human` | 사람이 직접 판단하거나 구현해야 하는 이슈        |
| 처리 안 함         | `wontfix`         | 이 저장소에서 처리하지 않기로 한 이슈            |
